原型

TInteger min(TInteger i, ... )
TDouble min(TDouble i, ... )
TInteger max(TInteger i, ... )
TDouble max(TDouble i, ... )

说明

求最小值，最大值，参数是整数或者浮点数，至少2个参数，最多30个参数。

参数

i

参与比较的整数或者浮点数

返回

最小值或者最大值

示例

var TInteger i;
var TDouble x;
i = min(10, 5, 8, 12, 4, 6);
assert(i == 4);
i = max(10, 5, 8, 12, 4, 6);
assert(i == 12);
x = min(10, 5.1, 8, 12, 4.2, 6);
assert(x == 4.2);
x = max(10, 5.1, 8, 12.2, 4, 6);
assert(x == 12.2);
