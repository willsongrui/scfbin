原型

TString TRadius::encode()

说明

对Radius消息进行编码

参数

无

返回

返回编码之后的完整的Radius消息可以直接发送

示例

var TRadius r;
var TString s, sr;
r = new TRadius;
r.setcommandcode(2);
r.setid(10);
r.setauth('1234567890123456'); // a string after md5 encryption
r.setcontent(s); // s is constructed by xmlsib
sr=r.encode();
udpsend(argv0,atoi(argv1),sr); // send the messagae
