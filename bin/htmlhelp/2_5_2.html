原型

TInteger TSCPTimer::timeout([eventProcessLabel])

说明

设置timer到达后要执行的业务逻辑片段的入口。
要求该业务逻辑片断:
1. 禁止中断
2. 结束后使用return 返回

参数

eventProcessLabel

业务逻辑片段入口对应的sib的标识

返回

示例

var TSCPTimer timer;
timer = new TSCPTimer;
timer.timeout([timeoutproc]);
timer.settimer(10);
...
on([timeoutproc], TSCPTimer t)
{
   // execute this part after 10 seconds
   // ...
}
