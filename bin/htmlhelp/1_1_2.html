原型

TInteger strlen(TString s)
TInteger binlen(TString s)
TInteger TString::length()

说明

求字符串长度。strlen考虑字符串结束符0，遇到认为结束；而binlen或者TString::length则返回二进制字符串的实际长度

参数

s

要求的长度的字符串

返回

字符串长度

示例

assert(strlen('12345') == 5);
assert(binlen('ABC\00ABC\00') == 8);
assert(strlen('ABC\00ABC\00') == 3); // the first \00 as the string end
