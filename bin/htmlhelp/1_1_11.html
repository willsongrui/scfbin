原型

TString::foreachbyte(TString operater, TInteger value)

说明

用于对一个字符串的每一个字节做一个运算，可以用于简单的字符串加密

参数

operater

操作符，支持加，减，乘，除，取余，与，或，异或

value

操作值

返回

0

示例

var TString s;
s.init(10, `A`);
assert(s == 'AAAAAAAAAA');
s.foreachbyte('+', 3);
assert(s == 'DDDDDDDDDD');
s.foreachbyte('-', 1);
assert(s == 'CCCCCCCCCC');
