原型

TINESObj TCommandLine::getinesobj()

说明

对于INES或者别的自动机调用子业务逻辑导致的TCommandLine对象，子业务逻辑可以调用这个函数来获得相关的TINESObj对象，然后利用这个对象来回送消息。
这个TINESObj对象不能使用delete函数删除

参数

无

返回

TINESObj对象的句柄

示例

void main(TCommandLine cmd)
{
   var TINESObj i;
   i=cmd.getinesobj();
   i.synctofsm('serviceevent a b');
   ...
}
