原型

TString tldntomscid(TString tldn)

说明

使用config.mscid配置，根据TLDN获取CDMA网络所对应的MSCID

参数

tldn

CDMA网络TLDN号，10进制字符串

返回

MSCID号码，16进制字符串

示例

void main()
{
   var TString tldn,mscid;
   tldn = '13334710';
   mscid = tldntomscid(tldn);
   assert(mscid == '37288A');
   tldn = '8613334710'
   mscid = tldntomscid(tldn);
   assert(mscid == '37288A');
   tldn = '013334710';
   mscid = tldntomscid(tldn);
   assert(mscid == '37288A');
   tldn = '13334710999';
   mscid = tldntomscid(tldn);
   assert(mscid == '37288A');
}
