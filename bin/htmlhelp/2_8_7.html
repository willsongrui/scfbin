原型

TInteger TFTPClient::listdir(TString dirname)

说明

列目录，需要在这个函数后再使用getlastinfo()才能获得文件名列表

参数

dirname

远端目录名

返回

示例

TFTPClient ftp;
TString fileList;
ftp = new TFTPClient;
ftp.init('ahost'，21，'user'，'password');
ftp.listdir('dirname');
fileList = ftp.getlastinfo(); // get the file list string
print(fileList)
