原型

TInteger TFTPClient::init(TString host, TInteger port, TString user, TString passwd)

说明

初始化TFTPClient对象，初始化后只是在对象内部保存这些值，并不真正的连接到远端FTP服务器

参数

host

主机名（或者IP地址）

port

端口号

user

用户名，匿名用户用'anonymous'

passwd

用户口令，匿名用户用'a@b.com'

返回

应该永远返回正确（即返回0）

示例

var TFTPClient ftp;
ftp = new TFTPClient;
ftp.init('ahost'，21，'user'，'password');
