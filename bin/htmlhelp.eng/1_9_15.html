SYNOPSIS

TString hxmlgetattr(TMXLTree handle, TString path, TString default)
TString TXMLTree::getattrvalue(TString path, TString default)

DESCRIPTION

To get the value of an attribute according to a path. The last node of the path is the attribute name

ARGUMENTS

handle

The handle of the TXMLTree object

path

The path of the attribute

default

The default value

RETURN VALUES

The attribute value

EXAMPLES

i = hxmlparse('<x>xxx<y cvd:z="aaa">yyy</y></x>');
s = hxmlgetattr(i, 'x/y/z', '');
s1 = i.getattrvalue('x/y/cvd:z','null');
s2 = i.getattrvalue('x/y/z','null');
assert(s == 'aaa');
assert(s1 == 'aaa');
assert(s2 == 'aaa');
