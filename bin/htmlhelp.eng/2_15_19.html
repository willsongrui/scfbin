SYNOPSIS

TInteger TTCAPDialogue::invokereq(TInteger opID, TInteger opType, TInteger timeout, TString para, TInteger linkedid=-1)

DESCRIPTION

To send the INVOKE request

ARGUMENTS

opID

integer. The operation code of the operation to be sent

opType

integer. The operation type:
1: Both success and failure reported
2: Only failure reported
3: Only success reported
4: Neither success nor failure reported

timeout

integer, The timer of the operation to be sent. Its unit is second

papa

the ASC string that is the parameter of the error to be sent. Its length must be even. For example '0330081010' indicates that the parameter is 5 bytes: 30 03 80 01 01

linkedID

integer. If this operation to be sent is the linked operation of another operation received, this argument is the invoke ID of the operation received. For example in INAP, If SRR is sent by calling this function, this argument is the invoke id of PA received before

RETURN VALUES

0
