SYNOPSIS

TString ntptimestamp(TInteger t)

DESCRIPTION

To get the NTP time stamp according to a seconds integer from 1970-1-1 00:00:00.

ARGUMENTS

t

A second integer. Normally it is the result of tosec() function

RETURN VALUES

A 4 bytes string

EXAMPLES

assert(tosec('20070723093100') == 1185154260);
assert(tosec2('20070723093100') == '3394143060');
assert(ntptimestamp(1185154260) == '\CA\4E\83\54');
assert(strh2u('CA4E8354') == '3394143060'); // 3394143060 = 1185154260 + 2208988800
