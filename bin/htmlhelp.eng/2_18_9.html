SYNOPSIS

TString TSCPTable::colname(TInteger colid)
TInteger TSCPTable::colid(TString colname)

DESCRIPTION

colname() is to get column name according to the column index. colid() is to get column index according to the column name

ARGUMENTS

colid

column index

colname

column name

RETURN VALUES

returned column name or index

EXAMPLES

var TSCPTable t;
t=new TSCPTable(3,4);
t[1][1]='11';
t[2][1]='21';
t[3][1]='31';
t[1][4]='14';
t.settitle(1,'A');
t.settitle(2,'B');
t.settitle(3,'C');
t.settitle(4,'D');
assert(t.colid('A') == 1);
assert(t.colname(3) == 'C');
