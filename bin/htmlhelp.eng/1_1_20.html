SYNOPSIS

TString TString::cuthead(TString sepFlag)

DESCRIPTION

To cut the head part of a string according to the separator. The cut part (not include the separator) is returned. If the separator cannot be found, the whole string will be returned and the source string is set to null

ARGUMENTS

sepFlag

The separator string

RETURN VALUES

The cut string

EXAMPLES

var TString s='123 456';
s1 = s.cuthead(' ');
assert(s == '456');
assert(s1 == '123');
s1 = s.cuthead('A');
assert(s == '');
assert(s1 == '456');
