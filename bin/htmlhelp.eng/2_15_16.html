SYNOPSIS

TInteger TTCAPDialogue::continuereq(TString userInfo='', TInteger type=0)

DESCRIPTION

To send the CONTINUE request

ARGUMENTS

userInfo

User information string. The default value is a null string

type

integer. The default value is 0 for normal BEGIN. 1 is for ConversationWithPermition, 2 is for ConversationWithoutPermition

RETURN VALUES

0
