SYNOPSIS

TString  str0cat(TString arg1, TString arg2)

DESCRIPTION

String concatenation. A null character will be added between the two strings

ARGUMENTS

arg1

The first string

arg2

The second string

RETURN VALUES

A new string after concatenation

EXAMPLES

var TString s = str0cat('aaa', 'bbb');
assert(s == 'aaa\00bbb');
