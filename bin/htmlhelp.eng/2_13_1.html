SYNOPSIS

TString TSipCallSegment::getid()
TInteger TSipCallSegment::setid(TString csID)

DESCRIPTION

To get or set the ID of a TSipCallSegment object

ARGUMENTS

csID

the Id to be set

RETURN VALUES

serid() returns 1 for success and 0 for failure. getid() returns the ID string

EXAMPLES

var TSipCallSegment cs;
var TSipMsg req;
var TString idValue;
...
cs = req.createcallsegment();
cs.setid(idValue);
idValue = cs.getid();
