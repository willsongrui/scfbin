SYNOPSIS

TString monthlastday(TString t1)

DESCRIPTION

To get the last day of the month that t1 indicated

ARGUMENTS

t1

The time string that format is YYYYMMDDHHmmSS

RETURN VALUES

A new time string that format is YYYYMMDDHHmmSS

EXAMPLES

var TString stmp;
stmp =monthlastday('20101020');
print(stmp);
本例返回时间：20101031142135
142135为系统本天当前时间
参数具体到秒时，返回的结果为查找月最后一天的相应时间。否则返回最后一天的与当前时间一样的时间
