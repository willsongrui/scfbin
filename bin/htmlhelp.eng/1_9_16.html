SYNOPSIS

TInteger hxmlelement(TString tag, TString content)

DESCRIPTION

To construct a simple TXMLTree object using a tag name and a content string

ARGUMENTS

tag

The tag name

content

The content string

RETURN VALUES

The constructed TXMLTree object handle

EXAMPLES

i = hxmlelement('aaa', 'bbb');
s = hxmlstr(i);
assert(s == '<aaa>bbb</aaa>');
