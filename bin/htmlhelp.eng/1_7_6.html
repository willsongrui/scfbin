SYNOPSIS

TInteger filetype(TString path)

DESCRIPTION

To get the file type

ARGUMENTS

path

The file path

RETURN VALUES

returns -1 for file not existed. Other values:
1: Named pipe
4: Directory
8: Normal file
10: Symbol link

EXAMPLES

var TString sfile;
sfile=sprintf('%s/log/test.txt',getenv('CINDIR'));
iresult = filetype(sfile);
print('file type ' + itoa(iresult));
