SYNOPSIS

TXMLTree TCommandLine::getxmltree()

DESCRIPTION

The TCommandLine object generated by the serviceEvent object may have a xml tree parsed from the message body. This function can get the xml object from the TComandLine object.
This xml tree cannot be deleted using delete function.

ARGUMENTS

no arguments

RETURN VALUES

handle of the xml tree object

EXAMPLES

var TXMLTree x;
on(serviceEvent, TCommandLine cmd)
{
   x=cmd.getxmltree();
   print(x.tostr());
}
