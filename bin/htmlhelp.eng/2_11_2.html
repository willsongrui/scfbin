SYNOPSIS

TString TSoap::tostr()

DESCRIPTION

To convert this TSoap object to a string. The string will include the xml process instruction and this is different from TXMLTree::tostr().

ARGUMENTS

no arguments

RETURN VALUES

EXAMPLES

var TSoap soapObj;
soapObj = new TSoap;
soapObj.init('', 1);
var TString ssoap;
ssoap = soapObj.tostr();
