SYNOPSIS

TDouble TDouble::sin()
TDouble TDouble::cos()
TDouble TDouble::tan()
TDouble TDouble::asin()
TDouble TDouble::acos()
TDouble TDouble::atan()

DESCRIPTION

To get the trigonometric value

ARGUMENTS

no arguments

RETURN VALUES

The trigonometric value

EXAMPLES

var TDouble d, d1;
d1 = d.sin();
d1 = d.cos();
d1 = d.tan();
d1 = d.asin();
d1 = d.acos();
d1 = d.atan();
