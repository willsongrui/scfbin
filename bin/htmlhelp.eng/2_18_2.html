SYNOPSIS

TSCPTable[][]=TString
TString=TSCPTable[][]

DESCRIPTION

To set or get the value of a table element. Integers in bracket are the index of rows and the index of columns. The first one is 1

ARGUMENTS

no arguments

RETURN VALUES

EXAMPLES

var TString s;
var TSCPTable t;
t = new TSCPTable(2,2);
t[1][2] = 'aaaaa';
s = t[1][2];
