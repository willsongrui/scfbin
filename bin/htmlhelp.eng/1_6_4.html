SYNOPSIS

TString sqlstrencode( TString s)
TString sqlstrdecode( TString s)

DESCRIPTION

sqlstrencode() is to encode some strings include special characters before constructing sql statement. sqlstrdecode() is to decode some string include special characters when processing sql select result. The special characters include enter or vertical line

ARGUMENTS

s

The source string

RETURN VALUES

The result string

EXAMPLES

assert(sqlstrencode('abc\ndef|123') == 'abc\\0Adef\\7C123');
assert(sqlstrdecode('abc\\0Adef\\7C123') == 'abc\ndef|123');
