SYNOPSIS

TInteger TTCAPDialogue::resultreq(TInteger invokeID, TInteger opID, TString para='')

DESCRIPTION

To send the RESULT-L request

ARGUMENTS

invokeID

integer. The invoke ID of the operation received

opID

integer, the operation code of the result to be sent

papa

the ASC string that is the parameter of the error to be sent. It's length must be even. For example '0330081010' indicates that the parameter is 5 bytes: 30 03 80 01 01

RETURN VALUES

0
