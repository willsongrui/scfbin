SYNOPSIS

TString  strxchar( TString arg1)

DESCRIPTION

Exchange each pair of character of a string. Normally, the length of the string is even. If the length is odd, character '0' will be appended before exchanging

ARGUMENTS

arg1

source string

RETURN VALUES

new string after exchanging

EXAMPLES

assert(strxchar('1234') == '2143')
assert(strxchar('123')=='2103');
assert(strxchar('1')=='01');
assert(strxchar('')=='');
