SYNOPSIS

TString nextweekday(TString t1, TInteger s)

DESCRIPTION

To get a new time string according to a time string and a next weekday index ( 0 for Sunday, 1 for Monday ... 6 for Saturday ).

ARGUMENTS

t1

The time string. The format is YYYYMMDDHHmmSS

s

weekday index, It is an integer between 0 and 6 ( 0 for Sunday, 1 for Monday ... 6 for Saturday )

RETURN VALUES

A new time string. The format is YYYYMMDDHHmmSS

EXAMPLES

// for example, 2002-5-20 is Monday
assert(nextweekday('20020520000000', 6) == '20020525000000'); // get next Saturday
assert(nextweekday('20020520000000', 5) == '20020524000000'); // get next Friday
