SYNOPSIS

TString TRadius::getvalue(tagpath, TString defaultValue)
TInteger TRadius::getvalue(tagpath, TInteger defaultValue)

DESCRIPTION

To get a parameter value of a radius message

ARGUMENTS

tagpath

the tagpath of the parameter

defaultValue

the default value if the parameter is not existed

RETURN VALUES

the integer or string value got

EXAMPLES

var TRadius r = new TRadius();
r.decode(s); // s a complete radius message from UDP

// get the string parameter
var TString s1=r.getvalue(<<radius.invoke.access-request.user-name>>,'null');
var TString s2=r.getvalue(<<radius.invoke.access-request.user-password>>,'null');
var TString s3=r.getvalue(<<radius.invoke.access-request.nas-ip-address>>,'null');

// get the integer parameter
var TInteger i1=r.getvalue(<<radius.invoke.access-request.nas-port>>,-1);

// get one parameter array element
var TString s5=r.getvalue(<<radius.invoke.access-request.framed-compression[1]>>,'null');
var TString s6=r.getvalue(<<radius.invoke.access-request.framed-compression[2]>>,'null');

// get parameter array elements number
i2=r.getvalue(<<radius.invoke.access-request.framed-compression>>,0);
