SYNOPSIS

TString  ltrim (TString s)
TString  rtrim (TString s)
TString  trim (TString s)

DESCRIPTION

Trim a string. Function ltrim delete the left space character of the string, Function rtrim delete the right space character of the string. Function trim not only delete the left and right space character but also replace the multi-space to one space in the string

ARGUMENTS

s

Source string to trim

RETURN VALUES

A new string after trim

EXAMPLES

assert(ltrim('  abc') == 'abc');
assert(rtrim(' abc  ') == ' abc');
assert(trim('  ab  cd  ') == 'ab cd');
