SYNOPSIS

TInteger TSCPTable::settitle(TInteger col,TString title)
TInteger TSCPTable::cleartitle()

DESCRIPTION

To set or unset the name of a column. If column names of a table is set, the tag of converted xml string will be this name

ARGUMENTS

col

column index, first is 1

title

column name

RETURN VALUES

0

EXAMPLES

var TSCPTable t;
t=new TSCPTable(3,4);
t[1][1]='11';
t[2][1]='21';
t[3][1]='31';
t[1][4]='14';
t.settitle(1,'A');
t.settitle(2,'B');
t.settitle(3,'C');
t.settitle(4,'D');
assert(t.toxml2() ==
'<table><record><A>11</A><B></B><C></C><D>14</D></record><record><A>21</A><B></B>
<C></C><D></D></record><record><A>31</A><B></B><C></C><D></D></record></table>');
