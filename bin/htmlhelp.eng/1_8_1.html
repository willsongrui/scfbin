SYNOPSIS

TString switchtranslate(TString s, TString  defaultValue, ... )

DESCRIPTION

To translate a string according to the match result

ARGUMENTS

s

The string to be matched

defaultValue

The default value. If no string is matched, this value is returned

...

The following arguments are always used in pairs. The former is matching string and the later is the translated string if matched.

RETURN VALUES

The translated string

EXAMPLES

s1 = 'ACR_RECORD';
s = switchtranslate(s1,'null','ACR_RECORD','AA','ACR_RECORD_01','BB');
assert(s == 'AA');
