SYNOPSIS

TInteger TTCAPDialogue::setac(TInteger acID)
TInteger TTCAPDialogue::getac()

DESCRIPTION

To get or set the application context of a dialogue

ARGUMENTS

acID

integer. -2 for not using application contex. Other values can be listed using ac command of scf

RETURN VALUES

getac() returns -2 for no AC, and returns other values can be listed using ac command of scf
