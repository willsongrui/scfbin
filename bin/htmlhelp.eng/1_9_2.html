SYNOPSIS

TString hxml(TXMLTree handle, TString path, TString default)
TString TXMLtree::getvalue(TString path, TString default)

DESCRIPTION

To get the content of an element

ARGUMENTS

handle

The handle of the TXMLTree object

path

The element path

default

The default value for path invalid

RETURN VALUES

The elememt content is returned if the path is valid. If the path is invalid the default value will be returned

EXAMPLES

var TXMLTree i = hxmlparse('<x cvd:attr1="hello">xxx<y>yyy</y></x>');
s1 = hxml(i,'x/y','null');
s2 = hxml(i,'x/z','null');
s3 = hxml(i,'x/cvd:attr1','null');
assert(s1 == 'yyy');
assert(s2 == 'null');
assert(s3 == 'hello');

s1 = i.getvalue('x/y','null');
s2 = i.getvalue('x/z','null');
s3 = i.getvalue('x/cvd:attr1','null');
assert(s1 == 'yyy');
assert(s2 == 'null');
assert(s3 == 'hello');
