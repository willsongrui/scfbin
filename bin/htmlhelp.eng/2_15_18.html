SYNOPSIS

TInteger TTCAPDialogue::abortreq(TString userInfo='', TInteger reason=0)

DESCRIPTION

To send the U-ABORT request

ARGUMENTS

userInfo

User information string. The default value is a null string

reason

integer. The reason for ABORT. The default value is 0
0: Unrecognized message type
1: Unreconnized transaction ID
2: Badly formatted transaction portion
3: Incorrect transaction portion
4: Transaction recource limitation

RETURN VALUES

0
