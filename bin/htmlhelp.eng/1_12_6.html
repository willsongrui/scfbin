SYNOPSIS

TDouble TDouble::ceil()
TDouble TDouble::floor()
TDouble TDouble::round()

DESCRIPTION

ceil() is to computes the smallest integral value not less than this TDouble value. floor() is to compute the largest integral value not greater than this TDouble value. round() is to compute the value round to this TDouble value.

ARGUMENTS

no arguments

RETURN VALUES

The computed result

EXAMPLES

var TDouble d = 1.6;
assert(d.ceil() == 2);
assert(d.floor() == 1);
assert(d.round() == 2);
