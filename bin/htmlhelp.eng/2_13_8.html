SYNOPSIS

TInteger TSipCallSegment::connect(TString calledNumber, TInteger legID, TString callingNumber)

DESCRIPTION

To connect to the called party

ARGUMENTS

calledNumber

called party number

legID

leg ID of called party (must be 2)

callingNumber

calling party number

RETURN VALUES

1 for success, 0 for failure

EXAMPLES

var TSipCallSegment cs;
var TSipMsg req;
...
cs = req.createcallsegment();
cs.connect ('9999',1,' ');
