SYNOPSIS

TInteger httpauthorization(TString a, TString nonce, TString method, TString password)

DESCRIPTION

This function is to do the authorization for server

ARGUMENTS

a

The content of Authenticate header of the message that the server received

nonce

nonce string

method

The method used to send message, 'GET' or 'POST'

password

The user password

RETURN VALUES

1 is for success, 0 is for failure
