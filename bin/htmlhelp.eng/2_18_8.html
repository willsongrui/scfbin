SYNOPSIS

TInteger TSCPTable::rowcount()
TInteger TSCPTable::colcount()

DESCRIPTION

To get the rows number and the columns number

ARGUMENTS

no arguments

RETURN VALUES

rows number or columns number

EXAMPLES

var TSCPTable t;
t=new TSCPTable(3,4);
assert(t.rowcount() == 3);
assert(t.colcount() == 4);
