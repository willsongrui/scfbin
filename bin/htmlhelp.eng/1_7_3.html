SYNOPSIS

TInteger fwrite(TString fileName, TInteger pos, TString str, TInteger length)

DESCRIPTION

To write a string to a file

ARGUMENTS

fileName

The file name

pos

The begin position of the file to write (-1 is for the end of the file)

str

The string to be written

length

The length to write. If the length is not valid ( less than or equal to 0, greater than the string length ), the whole string length is used

RETURN VALUES

The actual bytes number that is written

EXAMPLES

assert(fwrite('file',0,'good',-2) == 4);
