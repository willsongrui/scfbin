SYNOPSIS

TInteger strlen(TString s)
TInteger binlen(TString s)
TInteger TString::length()

DESCRIPTION

Get the length of a string. binlen and TString::length return the whole length of a binary string. strlen only counts the characters before the first null character

ARGUMENTS

s

The string

RETURN VALUES

The string length

EXAMPLES

assert(strlen('12345') == 5);
assert(binlen('ABC\00ABC\00') == 8);
assert(strlen('ABC\00ABC\00') == 3); // the first \00 as the string end
