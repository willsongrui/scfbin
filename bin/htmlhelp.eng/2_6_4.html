SYNOPSIS

TInteger TSCPChargeServer::stopcharge()

DESCRIPTION

To stop the charging

ARGUMENTS

no arguments

RETURN VALUES

EXAMPLES

var TSCPChargeServer ch1;
ch1 = new TSCPChargeServer;
ch1.addchargeclass(100);
ch1.setbalancesupervision(300, 0, 0);

ch1.onlimitreached([limitReach]);

ch1.onstop([recACR]);
ch1.startcharge();

sleep(1000);

ch1.stopcharge();
