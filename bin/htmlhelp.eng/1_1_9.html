SYNOPSIS

TString lower(TString s)
TString upper(TString s)

DESCRIPTION

Lowercase or uppercase converting. Function lower returns the string converted to all lowercase. Function upper returns the string converted to all uppercase

ARGUMENTS

s

Source string to be converted

RETURN VALUES

A new string after converted

EXAMPLES

assert(lower('Abc') == 'abc');
assert(upper('abC ') == 'ABC');
