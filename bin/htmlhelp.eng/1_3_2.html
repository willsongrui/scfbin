SYNOPSIS

TString gbktounicode(TString source)
TString unicodetogbk(TString source,TInteger length)

DESCRIPTION

gbktounicode() is to convert the GBK string to UCS2 string. unicodetogbk() is to convert the UCS2 string to GBK string

ARGUMENTS

source

The string to be converted

length

The string length

RETURN VALUES

The result string
