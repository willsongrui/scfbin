SYNOPSIS

TInteger udpsend(TString host, TInteger port, TString s)
TInteger  udpsendbyname(TString host, TInteger port, TString s, TString logicName)

DESCRIPTION

udpsend() is to send a UDP message and it can not indicate a sending party port number, and need to config the config.udp.*. udpsendbyname() can specify a sending port

ARGUMENTS

host

The destination host name

port

The destination port number

s

The string to be sent

logicName

The logic name that configured the UDP sending port. It is configured in the config.server.*, using udp protocol

RETURN VALUES

0 is for success and a negative value is for failure
