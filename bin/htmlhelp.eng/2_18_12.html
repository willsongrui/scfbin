SYNOPSIS

TSCPTable TSCPTable::diffbycol(TSCPTable table2, TInteger col1, TInteger col2)

DESCRIPTION

To compare a column with a column of another table

ARGUMENTS

table2

the handle of another table

col1

self column index

col2

another table column index

RETURN VALUES

comparison result. It is a new table which has only one column include the different rows of the two comparison columns.

EXAMPLES

var TSCPTable t, t2, t3;
t=new TSCPTable(3,4);
t[1][1]='11';
t[2][1]='21';
t[3][1]='31';
t[1][4]='14';
t2=new TSCPTable(2,2);
t2[2][1]='21';
t3=t.diffbycol(t2,1,1);
assert(t3.tostr('|') == '11|31');
