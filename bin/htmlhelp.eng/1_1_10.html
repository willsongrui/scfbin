SYNOPSIS

TString::foreachbyte(TString operater, TInteger value)

DESCRIPTION

To do an operation for each byte of a string. This function can be used for simple string encryption/decryption

ARGUMENTS

operator

operator, can be
'+' for addition
'-' for subtraction
'*' for multiplication
'/' for division
'%' for mod
'&' for AND
'|' for OR
or an up-arrow for XOR

value

operation value

RETURN VALUES

0

EXAMPLES

var TString s;
s.init(10, `A`);
assert(s == 'AAAAAAAAAA');
s.foreachbyte('+', 3);
assert(s == 'DDDDDDDDDD');
s.foreachbyte('-', 1);
assert(s == 'CCCCCCCCCC');
