SYNOPSIS

TInteger tracetext(TString s)
TInteger tracebin(TString s)

DESCRIPTION

To send some trace information to other process using UDP. If all characters in the string are printable, you can use print() function. If not, you must use printbin() function. The sending of these two functions can be switched off or switched on using 'display' command

ARGUMENTS

s

The content to be output

RETURN VALUES

0
