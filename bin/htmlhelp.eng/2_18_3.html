SYNOPSIS

TInteger TSCPTable::resize(TInteger row, TInteger col)

DESCRIPTION

To change the size of a table. Normally, if you assign a value to an element of the table, the table size can be automatically changed according to the index. But you can use resize() first for better performance.

ARGUMENTS

row

number of rows

col

number of columns

RETURN VALUES

0

EXAMPLES

var TString s;
var TSCPTable t;
t = new TSCPTable(2,2);
t.resize(10,10);
