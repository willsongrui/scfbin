SYNOPSIS

TString  sprintf(TString format£¬...)
TString  sprintbin(TString binStr)

DESCRIPTION

sprintf is to construct a new string according to the format and some arguments. It is very similar to the printf function in C language. sprintbin is to convert a string that may include non-printable characters to a printable string.

ARGUMENTS

format

Same as the format argument of sprintf function as C language

binStr

A string that may include non-printable characters

...

Integer or string, The maximum is 30

RETURN VALUES

A new string

EXAMPLES

var TString s1 = 'aaa';
var TString s2 = 'bbb';
var TString s = sprintf('%s%s',s1,s2);
assert(s == 'aaabbb');

var TString binStr = 'abc\00\00\01\01abc';
print(sprintbin(binStr));
// may output:
// 0     61 62 63 00 00 01 01 61  62 63                     abc....abc
