SYNOPSIS

TXMLTree hxmldupelement(TXMLTree handle, TString path)
TXMLTree TXMLTree::dupelement(TString path)

DESCRIPTION

To duplicate an element of a TXMLTree object and create a new TXMLTree element

ARGUMENTS

handle

The TXMLTree object handle

path

The path of the element to be duplicated

RETURN VALUES

A new TXMLTree object handle

EXAMPLES

i = hxmlparse('<x>xxx<y>yyy</y></x>');
j = hxmldupelement (i,'x/y');
s = hxmlstr(j);
assert(s == '<y>yyy</y>');
