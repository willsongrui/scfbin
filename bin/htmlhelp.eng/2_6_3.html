SYNOPSIS

TInteger TSCPChargeServer::startcharge()

DESCRIPTION

To start the charging after set the parameters

ARGUMENTS

no arguments

RETURN VALUES

EXAMPLES

var TSCPChargeServer ch1;
ch1 = new TSCPChargeServer;
ch1.addchargeclass(100);
ch1.onstop([recACR]); // For process the CDR
ch1.startcharge();
