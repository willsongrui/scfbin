SYNOPSIS

TDouble globaldis(TDouble x1, TDouble y1 ,TDouble x2, TDouble x2, TDouble r=6371.004)

DESCRIPTION

To calculate the distance of two point of a global

ARGUMENTS

x1

The longitude the first point

y1

The latitude of the first point

x2

The longitude the second point

y2

The latitude of the second point

r

The radius of the global. The unit is kilometer

RETURN VALUES

The distance that the unit is kilometer

EXAMPLES

assert(globaldis(116.4, 39.9, 121.4, 31.2, 6370) == 1067.273602186433);
