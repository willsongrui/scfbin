SYNOPSIS

TString THttpContent::getbuffer(TInteger isInlucdeHedaer=0)
TXMLTree THttpContent::getxmlbody()
TSoap THttpContent::getsoapbody()

DESCRIPTION

getbuffer() is to get the whole content and return a string.
If the Content-Type is Text/xml, getxmlbody() can be used to get the TXMLTree object according to the content.
If the Content-Type is Text/xml and it is a soap message, getsoapbody() can be used to get the TSoap object.

ARGUMENTS

isInlucdeHedaer

0 means not including the header. Values not equal to 0 means including the header. The default value is 0

RETURN VALUES

getbuffer() returns a string. getxmlbody() returns the handle of TXMLTree object. getsoapbody() returns the handle of TSoap object

EXAMPLES

contentObj.getbuffer();
contentObj.getsoapbody();
