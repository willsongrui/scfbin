SYNOPSIS

TInteger hxmlsetattr(TInteger i, TString path, TString attrName, TString attrValue)

DESCRIPTION

To set the value of an attribute of an element

ARGUMENTS

i

The handle of the TXMLTree object

path

The element path

attrName

The attribute name

attrValue

The new attribute value to be set

RETURN VALUES

0 for success and -1 for failure

EXAMPLES

i = hxmlparse('<x a="1">123</x>');
hxmlsetattr(i, 'x', 'a', '2');
s = hxmlstr(i);
assert(s == '<x a="2">123</x>');
