SYNOPSIS

TSipCallSegment TSipMsg::createcallsegment()

DESCRIPTION

After the service logic received the sip message, calling this function can construct a TSipCallSegment object to process the request.

ARGUMENTS

no arguments

RETURN VALUES

the handle of the TSipCallSegment object

EXAMPLES

void main(TSipMsg req)
{
   var TSipCallSegment cs;
   cs = req.createcallsegment();
   ...
}
