SYNOPSIS

TInteger hxmlccbytag(TXMLTree handle, TString path)
TInteger TXMLtree::childcountbytag(TString path)

DESCRIPTION

To get the element number that can match the path

ARGUMENTS

handle

The handle of the TXMLTree

path

The path

RETURN VALUES

Element number integer

EXAMPLES

i = hxmlparse('<a><b>b1</b><c>c1</c><d>d1</d><b>b2<c>cc1</c></b><c>c2</c></a>') ;
j = hxmlccbytag(i, 'a/b');
assert(j == 2);
