SYNOPSIS

THttpClient::THttpClient(TString logicName)

DESCRIPTION

To create the THttpClient object

ARGUMENTS

logicName

The logic name. This name don't need to configured in file config.client

RETURN VALUES

EXAMPLES

var THttpClient httpclientobj;
httpclientobj=new THttpClient('qc');

逻辑名'qc'不需要在config.client中配置。
