SYNOPSIS

TString htmlentity2gbk(TString source)
TString htmlentity2gbk(TString source)

DESCRIPTION

These functions is to replace the html entities in the source string to the GBK character or UTF8 character and return a new string. It will not be replaced if the html entity format is not valid or no GBK character defined (for htmlentity2gbk()). The 253 html entity names defined in HTML4.0 are supported.

ARGUMENTS

source

The string to be converted

RETURN VALUES

The result string

EXAMPLES

s = htmlentity2gbk('&amp;&gt;&&&&#x61&#x61;&#x54C8;');
assert(s == '&>&&&&#x61a\B9\FE', s);
s = htmlentity2gbk('&amp;&gt;&&&&#x61&#x61;&#x54C8;');
assert(s == '&>&&&&#x61a\E5\93\88', s);
