SYNOPSIS

TInteger tcpdeletelink(TString logicName, TInteger linkID, TString key)

DESCRIPTION

To delete a socket link

ARGUMENTS

logicName

The logic name. It is configured in the config.server.* or cofnig.client.* file

linkID

The link ID

key

The key correlated to the link. If it is a null string, only link ID is used.

RETURN VALUES

0 for success
