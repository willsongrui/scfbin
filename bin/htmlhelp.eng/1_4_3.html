SYNOPSIS

TString descrypt(TString arg1, TString arg2, TInteger arg3, TInteger arg4=0)
TString descbccrypt(TString arg1, TString arg2, TInteger arg3, TInteger arg4=0)

DESCRIPTION

descrypt() is to encrypt or decrypt a string using DES algorithm. descbccrypt() is to encrypt or decrypt a string using CBC based DES algorithm.

ARGUMENTS

arg1

The 8 bytes source string

arg2

The 8 bytes key

arg3

The length of source string

arg4

The mode. 0 is for encryption, other values is for decryption

RETURN VALUES

The 8bytes result string
