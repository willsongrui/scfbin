SYNOPSIS

TString hcode(TString number)

DESCRIPTION

To get the HCODE according to the config.hcode configuration

ARGUMENTS

number

The MSISDN number

RETURN VALUES

The HCODE

EXAMPLES

void main()
{
   var TString areaCode,msisdn;
   msisdn = '13910590001';
   areaCode = hcode(msisdn);
   ...
}
