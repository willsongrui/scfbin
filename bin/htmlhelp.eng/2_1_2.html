SYNOPSIS

TInteger TObject::setid(TString id)
TString TObject::getid()

DESCRIPTION

Other object can derive these two method of TObject. It is to identify an object and search an object according to the ID.

ARGUMENTS

id

The ID to set

RETURN VALUES

setid() returns 0 for success. getid() returns the object ID string

EXAMPLES

var TXMLTree x, y;
x = new TXMLTree()
x.setid('aaa');
assert(x.getid()=='aaa');
y = findobject(TXMLTree, 'aaa');
