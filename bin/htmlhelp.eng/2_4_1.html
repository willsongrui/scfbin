SYNOPSIS

TString THttpContent::getcontenttype()

DESCRIPTION

To get the value of 'Content-Type' field from the message header

ARGUMENTS

no arguments

RETURN VALUES

EXAMPLES


contObj.setbuffer('text/plain','<vxml version="1.0"><form id="form1"><block><prompt><audio src="10006.wav"/></prompt></block></form></vxml>');
var TString strtmp;
strtmp=contentObj.getcontenttype();
assert(strtmp=='text/plain');
