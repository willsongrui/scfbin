SYNOPSIS

TInteger TSipCallSegment::geterrorcase()

DESCRIPTION

To get the error case. This function is used for exception processing logic for synchronous etc(), pa(), pc(), pr()

ARGUMENTS

no arguments

RETURN VALUES

0: Normal
1: Default error
2: ETC failure
3: Time out
4: Collecting digits failure
5: Playing announcement failure
6: Recording failure

EXAMPLES

var TSipCallSegment cs;
var TSipMsg req;
...
cs = req.createcallsegment();
cs.geterrorcase ();
