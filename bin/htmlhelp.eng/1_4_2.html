SYNOPSIS

TString md5(TString s, TInteger length=0)

DESCRIPTION

To encrypt a string using MD5 algorithm.

ARGUMENTS

s

The string to be encrypted

length

if s include null character, you may use this argument to indicate the length. The default value is 0 that means only part before null character is used

RETURN VALUES

A 16 bytes string

EXAMPLES

assert(md5('abcd') == '\E2\FC\71\4C\47\27\EE\93\95\F3\24\CD\2E\7F\33\1F');
assert(md5('abcd\00abcd') == '\E2\FC\71\4C\47\27\EE\93\95\F3\24\CD\2E\7F\33\1F');
assert(md5('abcd\00abcd',9) == '\13\DA\E0\38\04\FD\7F\DA\C5\B0\8E\5B\41\02\82\72');
