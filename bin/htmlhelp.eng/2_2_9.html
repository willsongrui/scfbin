SYNOPSIS

TInteger THttpServer::sendresponse (THttpContent contentid)

DESCRIPTION

To send the response message

ARGUMENTS

contentid

The handle of the THTTPContent to be sent

RETURN VALUES

Values less than 0 are returned for failure. Other values are returned for success

EXAMPLES

var THttpContent cntObj;
cntObj=serverObj.getcontent();
serverObj.sendresponse(cntObj);
