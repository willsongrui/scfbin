SYNOPSIS

TInteger rematch(TString regexp, TString s)

DESCRIPTION

To check whether a string can match to a regular expression. The matching is for total string, not for sub string.

ARGUMENTS

regexp

The regular expression string

s

The string to match

RETURN VALUES

The matching result. 1 is returned for success. A negative integer is returned for matching failure and the absolute value indicate the failure position of the string. If the regular expression is invalid, an exception will be thrown out

EXAMPLES

var TString str='abcdefghijklmnop';
var TString regexp = '[a-z]{4,16}';
var TInteger ret=rematch(regexp, str);
assert(ret==1);
