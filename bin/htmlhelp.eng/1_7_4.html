SYNOPSIS

TString fread(TString fileName, TInteger pos, TInteger length)

DESCRIPTION

To read a string from a file

ARGUMENTS

fileName

The file name

pos

The begin position of the file to read

length

The length to read. If the length is equal to or less than 0, read to the end if the file

RETURN VALUES

The read result string. Notice: fread can only read file content less than 512K

EXAMPLES

s = fread('file', 0, 2)
