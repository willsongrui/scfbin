SYNOPSIS

TInteger []::index(TypeOfArrayMember mem)

DESCRIPTION

To get the index of an element of a array

ARGUMENTS

mem

element content of the array. The type of this argument must be the same as the array element. If the array is a TString array, the type of this argument must be TString. If the array is a TInteger array, the type of this argument must be a TInteger.

RETURN VALUES

integer, the index. The first one is 0. If the element cannot be found, -1 will be returned

EXAMPLES

var TInteger i[] = {1,2,3};
assert(i.index(1) == 0);
assert(i.index(2) == 1);
assert(i.index(5) == -1);
var TString s[] = {'aa','bb','cc'};
assert(s.index('aa') == 0);
assert(s.index('bb') == 1);
assert(s.index('ddd') == -1);
