SYNOPSIS

TInteger THttpContent::writefile(TString filename)
TInteger THttpContent::readfile(TString filename)

DESCRIPTION

writefile() is to write the binary file to the path cin/webxml. It dump the THttpContent object to the file.
readfile() is to read the binary file from the path cin/webxml and construct the THttpContent object according to the file content

ARGUMENTS

filename

The file name

RETURN VALUES

1 is returned for success. 0 is returned for failure

EXAMPLES

contObj=new (THttpContent);
......
contSubObj.setbuffer('text/plain','<vxml version="1.0"><form id="form1"><block><prompt><audio src="10006.wav"/></prompt></block></form></vxml>');
itmp=contObj.insertsubbody('multipart/related',contSubObj);
......
contObj.writefile('test.bin');
contObj.readfile('test.bin');
contObj.getcontenttype();
