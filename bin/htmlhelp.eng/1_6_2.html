SYNOPSIS

TString sqlgetstrprefix(TString s)

DESCRIPTION

To get a string include all prefix of another string. The result is used to construct a sql statement for 'in()' operation

ARGUMENTS

s

The source string

RETURN VALUES

A set of prefix of a string. Each prefix is in single quotation marks and commas separate them

EXAMPLES

assert(sqlgetstrprefix('12345') ==
'\2712345\27,\271234\27,\27123\27,\2712\27,\271\27'
// \27 is ', eg, '12345','1234','123','12','1'
