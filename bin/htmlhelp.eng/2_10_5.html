SYNOPSIS

TString TASNObject::getcode()

DESCRIPTION

To get an encoded ASC string of this object

ARGUMENTS

no arguments

RETURN VALUES

For example '0330081010' is returned for 5 bytes ASN.1 code: 30 03 80 01 01

EXAMPLES

var TASNObject a;
var TString s;
a = new TASNObject(48,'08500212345670');
s = a.getcode();
assert(s == '08500212345670');
