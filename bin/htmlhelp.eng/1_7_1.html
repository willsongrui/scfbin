SYNOPSIS

TString fgets(TString fileName, TInteger mode=0, TInteger flag=0)

DESCRIPTION

To read a file line by line. If the file name is same as the last calling of this function, the next line of this file is returned. If the file name is changed, another new file is open and the first line is returned.

ARGUMENTS

fileName

The file name

mode

The mode of opening file. 0 is for read only, values greater than 0 are for read and write. The default value is 0

flag

The flag for processing file end. The default value is 0 for return a null string if file is end

RETURN VALUES

A string of one file line including the enter character

EXAMPLES

var TString fileName,line;
fileName=...;
line=fgets(fileName,0,1);
while(line!='')
line=fgets(fileName,0,1);
print('file is end');
