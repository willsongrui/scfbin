SYNOPSIS

TInteger assert(TInteger boolExpression, TString userInfo='')

DESCRIPTION

To assert that a boolean expression is true. If the expression is true, this function will do nothing. If the expression is false, this function will output the 10949 warning information to the screen and the SCF.INFO.* file.

ARGUMENTS

boolExpression

The boolean expression to be asserted

userInfo

The information to be output if the expression is false

RETURN VALUES

0
