SYNOPSIS

TString strreplace(TString source, TString from, TString to)
TInteger TString::replace(TString from, TString to)

DESCRIPTION

To find a sub-string from a string and replace all found sub-string to a new string. The finding is case sensitive and support string including character 0. strreplace() function dose not change the source string and returns a new string. TString::replace() changes the source string

ARGUMENTS

source

source string

from

the sub-string to be found

to

the objective sub-string replaced

RETURN VALUES

strreplace() returns a new string. TString::replace returns 0

EXAMPLES

assert(strreplace('abcdABcd','ab','cc') == 'cccdABcd');
s='111111';
s.replace('1', '2');
assert(s == '222222');
s.replace('2', '\00\01');
assert(s == '\00\01\00\01\00\01\00\01\00\01\00\01');
s.replace('\01', '\02\03');
assert(s == '\00\02\03\00\02\03\00\02\03\00\02\03\00\02\03\00\02\03');
