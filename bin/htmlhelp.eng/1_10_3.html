SYNOPSIS

TInteger tcpconnectsend(TString logicName, TString host, TInteger port, TString s, TInteger length)

DESCRIPTION

To connect a socket server according to the host and port and send a message to the server

ARGUMENTS

logicName

The logic name. It is configured in config.client.* file

host

The host name. If it is a null string, the host name and port configured in the config.client.* file will be used.

port

The port number

s

The string to be sent

length

The length to be sent. If the value is 0, the whole s will be sent and one byte that the value is 0 will be sent for string end. If the value is greater than the length of s, the whole s will be sent

RETURN VALUES

If success, the new link ID will be returned. A negative is returned for failure
