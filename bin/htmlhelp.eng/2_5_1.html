SYNOPSIS

TInteger TSCPTimer::settimer( TInteger t)

DESCRIPTION

To set the timeout duration and start the timer.

ARGUMENTS

t

The time duration to be set. Values equal to or less than 0 are for reset the timer (same as TSCPTimer::clean).

RETURN VALUES

EXAMPLES

var TSCPTimer timer;
timer = new TSCPTimer;
timer.timeout([timeout]);
timer.settimer(10);
